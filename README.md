# 🌍 Chennai Air Quality Monitoring Dashboard

A simple web-based dashboard to visualize air pollution levels across different locations in Chennai.
It shows AQI, PM2.5, PM10, and CO₂ levels with interactive charts and a modern UI.

---

## 🚀 Features

* 📊 Pollution data visualization (simulated)
* 🌫️ AQI calculation based on PM2.5
* 📍 Multiple locations in Chennai
* 📈 Interactive charts using ECharts
* 🧊 Clean glassmorphism UI
* 🔍 Detailed view for each location

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* ECharts
* Font Awesome

---

## 📌 How It Works

* Pollution data is stored in JavaScript

* AQI is calculated using:

  AQI = PM2.5 × 3

* Data is displayed as:

  * Location cards
  * Comparison chart
  * Detailed charts
